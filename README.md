# Project Overview

This repository features an interactive R Shiny dashboard engineered to investigate and visualize the core parameters that drive a movie's critical and commercial success. Utilizing a cleaned and curated IMDb dataset of 5,487 contemporary films released between 2000 and 2020 , the project evaluates performance using a custom normalized success metric that balances average audience ratings, financial profit, release age, and user review volumes. Designed primarily as a decision-making tool for cinema production companies, the platform allows users to explore data-driven insights to strategically plan future film projects.

# Key Features

The application empowers users to filter data by genre, director, and cast to receive dynamically updated recommendations across five distinct tabs: Genre, Duration, Crew, Financial, and Trends. Key capabilities include extracting thematic motifs through genre-specific word clouds , determining optimal movie runtimes , and generating advanced network visualizations that highlight the most lucrative historical actor-director collaborations. Furthermore, the dashboard provides a specialized financial overview of budgeting vs. profitability , temporal analysis for peak seasonal release windows , and trend forecasting to predict how specific genres or talent pools will perform in future market landscapes.
